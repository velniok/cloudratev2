# CloudRate v2

> Веб-приложение для оценки музыки российских андерграунд-исполнителей с платформы SoundCloud.

🔗 **[Live Demo](https://cloudratev2.vercel.app/)** <!-- замени на свою ссылку -->

---

## О проекте

CloudRate — это платформа, где пользователи могут открывать для себя музыку российского андерграунда, оценивать треки и оставлять отзывы. Это вторая версия приложения, переписанная с нуля на современном стеке с переработанной архитектурой и структурой проекта.

---

## Стек технологий

### Frontend
- **React + Vite** + **TypeScript**
- **Redux** — глобальное управление состоянием
- **Axios** — HTTP-запросы
- **SCSS Modules** — стилизация компонентов
- **Feature-Sliced Design (FSD)** — архитектурная методология

### Backend
- **Node.js** + **Express** + **JavaScript**
- **PostgreSQL** — реляционная база данных
- **Cloudinary** — облачное хранилище для изображений
- **bcrypt** — хэширование паролей
- **JWT** — аутентификация на основе токенов

### Деплой
- **Vercel** — деплой клиента и сервера

---

## Функциональность

### ✅ Реализовано
- Регистрация и авторизация пользователей (JWT + bcrypt)
- Профили пользователей с возможностью редактирования
- Загрузка и обновление аватарок через Cloudinary

### 🚧 В разработке
- Страницы артистов и треков
- Оценка треков с необязательным текстовым отзывом
- Админ-панель:
  - Управление треками, артистами и пользователями (CRUD)
  - Общая статистика приложения

## Архитектура

Фронтенд построен по методологии **Feature-Sliced Design**, что обеспечивает чёткое разделение ответственности, масштабируемость и удобство поддержки кода.

```
client/
├── app/          # инициализация приложения, провайдеры
├── pages/        # страницы
├── widgets/      # самостоятельные блоки интерфейса
├── features/     # пользовательские сценарии
├── entities/     # бизнес-сущности (User, Track, Artist)
└── shared/       # UI-кит, утилиты, api
```
